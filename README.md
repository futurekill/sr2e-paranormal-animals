# Shadowrun 2E: Paranormal Animals of Europe

A content module for the [Shadowrun 2nd Edition system](../sr2e-foundryvtt) that
adds the Awakened animals from *Paranormal Animals of Europe* (FASA 7112) as
ready-to-drop **critter actors**.

- Each critter is an `npc` actor (race "critter") with full attributes, Reaction
  + initiative dice, attacks, and its Powers/Weaknesses summarised in the bio.
- Stat blocks only — the rules text and naming/expression appendices are not
  imported. Critters already in the system's core `critters` pack are not duplicated.

Requires the `sr2e` system (≥ 0.10.0). Personal table use only.
