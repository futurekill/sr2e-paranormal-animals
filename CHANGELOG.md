# Changelog

## 0.1.0 — Bestiary

First release. The complete *Paranormal Animals of North America* (FASA 7112)
bestiary as SR2E critter actors — **59 creatures** (A through Appendix I faerie
critters), every stat block verified against the page renders, built for the `sr2e`
system.

The setting/flavour text is not imported — game statistics only.
